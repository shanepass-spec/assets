# Scout Privacy Policy

**Last updated: April 25, 2026**

Scout is a personal Custom GPT operated by Shane Pass. This policy explains how Scout handles user input.

## What Scout collects

Scout receives only the messages users type into the ChatGPT interface during a conversation. Scout does not request, store, or transmit personally identifiable information beyond what users voluntarily share in chat.

## How conversations are processed

Conversations are processed by OpenAI's ChatGPT platform under OpenAI's privacy policy: https://openai.com/policies/privacy-policy

## Custom Actions

Scout uses one Custom Action that calls a public read-only endpoint hosted on Cloudflare Workers. This endpoint returns publicly available context information. No user data is written, stored, or shared by this Action.

## Data retention

Scout does not maintain its own database, user accounts, or memory of past conversations beyond what ChatGPT itself provides.

## Third-party services

Scout relies on:
- OpenAI / ChatGPT (conversation processing)
- Cloudflare Workers (Custom Action endpoint hosting)
